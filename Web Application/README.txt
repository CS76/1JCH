		
						        1JCH - PREDICTION
		
=======================================================================================
This folder contains the source code and .war file of the 1JCH-Web Application.
=======================================================================================
To run the web application 

1) Copy the jch.war file in to the tomcat %CATALINA_HOME%\webapps folder.

2) Start the tomcat server 

	On Windows:
      %CATALINA_HOME%\bin\startup.bat  (or)  %CATALINA_HOME%\bin\catalina.bat start
    
	On *nix:
      $CATALINA_HOME/bin/startup.sh  (or)  $CATALINA_HOME/bin/catalina.sh start

3) After startup, the 1jch-prediction web applications included with Tomcat will be
      available by visiting:
	  
      http://localhost:8080/jch
